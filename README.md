# ip_ltx

Python-скрипты для работы с конфигами формата **ltx** (*S.T.A.L.K.E.R.: Тень Чернобыля*), а также ряд утилит для моддинга на их основе.

В первую очередь были созданы как вспомогательная утилита для разработки модификации **Истинный путь** (v2.0, v2.1, v3.0), но основная часть функциональности может быть использована и для любого другого мода.

Совместимо с X-Ray Engine SoC (Тень Чернобыля) и OGSR Engine. На других движках часть функциональности может работать "из коробки" или при дополнительной настройке конфигов программы.

Описание возможностей, руководство по использованию и более детальная документация будут добавлены позже...
